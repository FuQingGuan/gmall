# 谷粒商城

> 谷粒商城是 尚硅谷的一套电商项目, 包括前台商城系统以及后台管理系统, 基于 SpringCloud + SpringCloudAlibaba + MyBatis-Plus实现, 前台商城系统包括：用户登录、注册、电商首页、商品搜索、商品详情、购物车、下订单流程、等模块。后台管理系统包括：系统管理、商品系统、优惠营销、库存系统、订单系统、用户系统、内容管理等七大模块。
>
> 该平台采用微服务架构，包括商品服务、订单服务、用户服务、搜索服务、配置中心、注册中心等多个微服务组件，可以支持高并发、高可用性的在线购物体验。
